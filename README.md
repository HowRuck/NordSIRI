## NordSIRI

A modern application built with Spring Boot and Angular that scrapes and processes real-time rail data using the SIRI (Service Interface for Real-Time Information) standard

> **📝 Note:** This project is primarily intended for educational purposes. It serves as a platform for me to refresh and strengthen my skills in Spring Boot, Java development, and the DevOps stack. As such, the code in this repository may not be optimized for production, performance, or architectural best practices.
