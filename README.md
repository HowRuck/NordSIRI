## GTFSynq 🚂

A modern application built with Spring Boot and Angular that processes GTFS (General Transit Feed Specification) data.

> **📝 Note:** This project is primarily intended for educational purposes. It serves as a platform for me to refresh and strengthen my skills in Spring Boot, Java development, and DevOps stack. As such, the code in this repository may not be optimized for production, performance, or architectural best practices.
